# 🤖 Responsive React Chatbot UI

A responsive and modern **chatbot user interface** built with **React**, **Tailwind CSS**, and **Vite**. It simulates a real-time chat experience with a clean design, dark/light theme toggle, and typing indicators.

---

## ✨ Features

- 💬 Chat UI with user & bot messages
- 🌗 Dark/Light mode toggle
- ⌨️ Typing indicator animation
- 📱 Fully responsive layout
- 🔄 Auto-scroll to latest message
- 🧠 Fake bot response (demo logic)

---

## 🧰 Tech Stack

- ⚛️ React
- 💨 Tailwind CSS
- ⚡ Vite (for blazing fast dev)
- 📁 CSS Modules
